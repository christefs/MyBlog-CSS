Proyectos

Clon de Wikipedia

Este proyecto es un duplicado visual de la página principal de Wikipedia a base de HTML y CSS con responsive design que reubica algunos elementos a partir de 900px.

Biblioteca Digital

Este proyecto trabaja el portal web de una biblioteca utilizando el framework Flask con base en python, a través del cual se realizan constantes consultas a una base de datos en PostgreSql para consultar usuarios y agregar otros nuevos y a su vez para que los usuarios autorizados puedan hacer consultas de una gran base de libros por título, autor entre otros y el sistema devuelva todas las posibles coincidencias.

Tienda Online

Este proyecto crea una tienda online con React donde se importan múltiples imágenes aleatorias desde una API y las que sean seleccionadas van siendo agregadas a un carrito de compra mientras se incrementa la cantidad y el precio.
